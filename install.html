1. Dein persönlicher SmartFlow-Code

Kopiere den folgenden Code und füge ihn auf deiner Website direkt vor </body> ein. Der Einbau dauert weniger als 2 Minuten.

  src="https://smartflow-automations.de/embed/v1/embed.js"
  data-client="sf_0001"
  async>
</script>
Kein Login. Keine Konfiguration. Nach dem Einfügen ist SmartFlow sofort aktiv.
